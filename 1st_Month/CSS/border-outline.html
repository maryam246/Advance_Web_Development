CSS Border & Outline Combination

This example demonstrates how to combine CSS border and outline properties to create visually appealing effects.

This element has both a border and an outline.

Explanation:

  • border: Adds a border around the element. You can control the width, style (solid, dashed, dotted, etc.), and color.
  • outline: Draws a line *outside* the border. It doesn't affect the element's dimensions. You can also control the width, style, and color.
  • outline-offset: This property adds space between the border and the outline, making them visually distinct. A positive value moves the outline further away from the border; a negative value moves it closer (potentially overlapping).
