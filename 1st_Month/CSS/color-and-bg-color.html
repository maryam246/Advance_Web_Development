CSS Color & Background Color

CSS properties like color and background-color are used to set the text color and background color of elements.

Color Values

  • Color names (e.g., red, blue)
  • Hex codes (e.g., #ff0000)
  • RGB values (e.g., rgb(255, 0, 0))
  • RGBA values (e.g., rgba(255, 0, 0, 0.5))
